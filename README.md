# Coffee Export Revenue Maximizer & EUDR Compliance Engine

---

## Project Overview

This project is an End-to-End Enterprise Data Solution designed to optimize the coffee export value chain.

By integrating live market data from the New York Coffee Exchange (ICE) with internal station-level processing data, the system:

- Identifies revenue leaks  
- Calculates regulatory impacts (NBE Forex Surrender)  
- Engineers strategies to capture high-margin European premiums  

---

## The Problem

Ethiopian coffee exporters face three critical "Value Killers":

### 1. Trapped Liquidity
- 93 Lots ($15.7M Gross) rejected due to quality defects  
- Results in $7.69M in retained FOREX being locked  

### 2. The "Silent Killer" – Physical Shrinkage
- 1.5% loss during transport (washing stations to Djibouti Port)  
- Leads to >$1M annual revenue loss  

### 3. Regulatory Risk
- Upcoming EUDR (European Union Deforestation Regulation)  
- Threatens access to premium EU markets (Germany, Italy)  

---

## The Solution (Data-Driven Strategy)

A four-phase analytical engine was developed:

### 1. Live Market Synchronization
- Real-time integration of KC=F (Coffee C ticker) 
- +$0.50 Quality Differential  
- Ensures audit-grade FOB pricing

### 2. Advanced Cost-Buildup Modeling
- Inland logistics: $0.18/kg  
- Admin fees + operational costs  
- Applies 0.985 shrinkage multiplier  
- Calculates True Net Realizable Value (NRV) 

### 3. EUDR Revenue Optimization
- Predictive model to identify compliant lots  
- Enables diversion to EU markets  
- Captures:
  - 22% specialty uplift
  - $0.50/kg premium

### 4. Executive Command Center
- Built using Streamlit dashboard
- Features:
  - Region/Station filtering  
  - High-value lot identification  
  - Re-inspection prioritization
